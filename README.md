# 케이슬립케어 개인정보 동의서

수미헬스의 케이슬립케어 프로그램 참여를 위한 개인정보 수집·이용 동의서입니다.

## 배포 방법

### Vercel CLI 사용:
```bash
cd ksleep-consent-form
vercel --prod
```

### Vercel 웹사이트 사용:
1. https://vercel.com 접속
2. "Add New" → "Project" 클릭
3. GitHub 연동 또는 직접 업로드
4. Deploy 클릭

## 파일 구조

- `index.html` - 동의서 메인 페이지
- `vercel.json` - Vercel 배포 설정

## 사용 방법

배포 후 생성된 URL을 환자에게 전송하면, 환자가 해당 링크에 접속하여 동의서를 작성하고 제출할 수 있습니다.

예: `https://ksleep-consent.vercel.app`
